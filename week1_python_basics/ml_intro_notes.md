# Introduction to Machine Learning

Machine Learning helps computers learn from data.

- In learning the data comes with labels.

- Unsupervised learning is when the model finds patterns on its own.

- Reinforcement learning is different the model learns from rewards and feedback.

- Machine Learning is used in many areas, like recommendation systems detecting fraud recognizing images and solving prediction problems.

- A basic Machine Learning process includes collecting data cleaning it training a model testing it and evaluating how well it works.
