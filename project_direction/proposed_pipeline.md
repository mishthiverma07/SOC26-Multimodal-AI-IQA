# Proposed Pipeline

1. Collect industrial product images.
2. Preprocess images using resizing, normalization, and augmentation.
3. Train or use a computer vision model for defect classification or detection.
4. Use CNN or transfer learning for image-based defect detection.
5. Store inspection documents or guidelines in a vector database.
6. Use RAG to retrieve relevant text information.
7. Use an LLM to generate a final explanation or report.
8. Combine image model output and text-based explanation in one interface.
