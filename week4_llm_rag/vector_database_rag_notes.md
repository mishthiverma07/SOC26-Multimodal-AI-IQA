# Vector Databases and RAG Notes

- Embeddings convert text into numerical vectors.
- Similar text chunks are close in vector space.
- Vector databases store embeddings for fast semantic search.
- ChromaDB is one example of a vector database.
- Cosine similarity finds the most relevant chunks.
- RAG stands for Retrieval Augmented Generation.
- RAG retrieves relevant context from documents and gives it to the LLM.
- RAG helps answer questions from PDFs and text files.
- Sources are important because LLMs can hallucinate.
