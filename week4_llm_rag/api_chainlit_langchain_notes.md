# API, Chainlit, and LangChain Notes

- Prompting means giving input instructions to an LLM.
- Completion is the output generated by the model.
- System message controls assistant behavior or personality.
- Temperature controls randomness and creativity.
- API key is used to access the model through code.
- Chainlit helps build chat-based user interfaces.
- LangChain helps connect prompts, LLMs, chains, retrievers, and tools.
- Streaming shows the response token by token.
- PromptTemplate helps create reusable prompt formats.
