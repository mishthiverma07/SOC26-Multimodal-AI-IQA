# LLM Introduction Notes

- LLM stands for Large Language Model.
- LLMs are trained on huge text datasets.
- They learn using next-token prediction.
- Tokenization converts text into numerical tokens.
- During inference, the model predicts one token at a time.
- Pre-training gives the model general knowledge.
- Fine-tuning helps convert the base model into a helpful assistant.
- RLHF uses human feedback to improve model behavior.
- LLMs can hallucinate, so outputs should be verified.
