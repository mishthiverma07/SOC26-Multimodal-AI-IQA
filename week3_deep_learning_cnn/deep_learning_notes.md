# Deep Learning Notes

- Deep learning uses neural networks with multiple layers.
- Neural networks contain input layers, hidden layers, and output layers.
- A neuron takes inputs, weights, bias, and applies an activation function.
- Activation functions introduce non-linearity.
- Common activation functions include sigmoid, tanh, ReLU, and leaky ReLU.
- Loss function measures prediction error.
- Gradient descent updates weights to reduce loss.
- Backpropagation uses chain rule to calculate gradients.
- TensorFlow and Keras are used to build deep learning models.
