# CNN Notes

- CNN stands for Convolutional Neural Network.
- CNNs are mainly used for image classification and computer vision.
- Filters act as feature detectors.
- Convolution creates feature maps.
- Padding helps preserve image size and gives corner pixels more importance.
- Stride controls how much the filter moves at a time.
- Pooling reduces image size and computation.
- ReLU adds non-linearity.
- CNN has feature extraction and classification parts.
- Data augmentation helps reduce overfitting by creating transformed image samples.
- Transfer learning uses pre-trained models for faster and better training.
