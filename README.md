# SoC 2026: Multi-Modal AI System for Industrial Quality Assurance

This repository contains my learning progress and implementation work for the Summer of Code 2026 project: Multi-Modal AI System for Industrial Quality Assurance.

The project focuses on understanding how machine learning, deep learning, computer vision, and large language models can be used together for industrial quality inspection and automated defect analysis.

## Progress till Mid-Term

### Week 1: Python Basics
- Learned Python fundamentals including variables, data types, strings, lists, dictionaries, loops, and functions.
- Understood basic machine learning concepts and types of ML.

### Week 2: Machine Learning Basics
- Studied supervised and unsupervised learning.
- Covered regression, classification, clustering, model evaluation, regularization, PCA, and ensemble learning.
- Practiced basic data handling using Pandas.

### Week 3: Deep Learning and Computer Vision
- Learned neural networks, TensorFlow/Keras, activation functions, loss functions, gradient descent, and backpropagation.
- Studied CNNs for image classification.
- Covered convolution, padding, stride, pooling, data augmentation, transfer learning, object detection, YOLO, and computer vision datasets.

### Week 4: LLMs and RAG
- Learned basics of Large Language Models, tokenization, pre-training, fine-tuning, RLHF, and prompt engineering.
- Explored API basics, Chainlit, LangChain, vector databases, embeddings, and document-based question answering using RAG.

## Project Direction

The final project idea is to build a multi-modal AI pipeline where:
- Computer vision models can detect visual defects in industrial images.
- LLM/RAG systems can help explain defect reports or retrieve relevant inspection guidelines.
- The system can combine image-based analysis with text-based reasoning.
