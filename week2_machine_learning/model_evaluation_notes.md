# Model Evaluation Notes

- Model evaluation helps check how well a model performs.
- Train-test split is used to test models on unseen data.
- Accuracy measures correct predictions out of total predictions.
- Precision tells how many predicted positives were actually correct.
- Recall tells how many actual positives were detected correctly.
- F1 score balances precision and recall.
- Confusion matrix helps understand classification errors.
- Cross-validation gives a more stable performance estimate.
